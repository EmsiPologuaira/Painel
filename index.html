📋 Sobre este Sistema

O Painel EMSI — Polo Base Guaíra é um sistema de informação em saúde indígena que trata dados pessoais sensíveis de indígenas atendidos pelo DSEI Litoral Sul / SESAI / Ministério da Saúde.

⚖️ Base Legal

O tratamento dos dados é fundamentado no Art. 7º, III e Art. 11º, II, "f" da LGPD (Lei 13.709/2018), que autoriza o tratamento de dados sensíveis para a tutela da saúde, exclusivamente por profissionais da saúde, serviços de saúde ou autoridade sanitária.

🛡️ Dados Tratados

  • Dados de identificação (nome, CPF, Cartão SUS, data de nascimento)
  • Dados de saúde (diagnósticos, atendimentos, vacinas, prontuários)
  • Dados socioeconômicos (situação familiar, benefícios sociais)
  • Dados de localização (aldeia, município)

📌 Suas Obrigações

  • Utilizar os dados exclusivamente para finalidades de saúde e assistência social indígena
  • Não compartilhar credenciais de acesso com terceiros
  • Não exportar ou copiar dados para fins pessoais
  • Comunicar imediatamente qualquer incidente de segurança ao Coordenador
  • Respeitar o sigilo profissional e o sigilo dos dados de saúde

⚠️ Penalidades

O uso indevido dos dados pode configurar infração à LGPD (multa de até R$ 50 milhões), além de responsabilização civil e criminal nos termos da legislação brasileira.
